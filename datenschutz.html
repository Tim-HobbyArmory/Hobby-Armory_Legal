2. Types of Data Processed

When using Hobby Armory, the following data may be processed:

  • Achievements / milestones: Progress data for achievements and goals
  • Goals & preferences: Monthly goals and user preferences
  • Images (user uploads): Photos of painted miniatures uploaded by you
  • Technical / usage data: App version, device/OS information and diagnostic data for stability and crash reporting

4. Data Storage Locations and Processors

Google Firebase (Google Cloud Platform): Authentication (Google Sign-In), user profiles, collection data, activity history, achievements, and preferences are stored using Firebase services (EU region). Data is encrypted in transit and at rest.

Cloudflare R2 (EU): Image files you upload are stored in Cloudflare R2 object storage (EU). Images may be delivered via CDN for performance.

Cloudflare Workers: Used to process image uploads/deletions (e.g., resizing/optimization workflows).

Transactional email: We send transactional emails (e.g., approval/rejection notifications) via Firebase/Google email infrastructure (extensions). No marketing emails are sent unless you explicitly opt in.

5. Images: Private vs. Community Catalog

Hobby Armory supports two image categories:

  • Private images: Photos you upload for your personal collection. These are intended to be visible only to you.
  • Community catalog images: Photos you voluntarily submit for the public catalog. Submissions are reviewed by an admin. Only approved images become publicly visible to all users.

Image processing: Uploaded images may be automatically resized and optimized for storage and delivery.

8. Data Retention
